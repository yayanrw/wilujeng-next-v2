# SimplePOS Pro — Project Overview

**SimplePOS Pro** adalah aplikasi Point of Sale (POS) berbasis web untuk toko retail modern. Fokus pada kecepatan transaksi kasir (barcode scanner), fleksibilitas harga (tiering/grosir), manajemen stok akurat, dan sistem loyalitas pelanggan.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js (App Router), Tailwind CSS, shadcn/ui |
| State | Zustand (cart, UI preferences, global state) |
| Backend | Next.js API Route Handlers |
| Database | Supabase (PostgreSQL) |
| ORM | Drizzle ORM |
| Auth | Better Auth |
| Cache | Upstash Redis |
| Hardware | Barcode Scanner (keyboard emulation), Thermal Printer (Web Print API) |
| Deployment | Vercel |
